O QUE É SISTEMAS DE INFORMAÇÃO?

Profissão do Futuro

Sistemas de Informação é o curso que conecta tecnologia, pessoas e negócios. Você aprende a criar soluções digitais, gerenciar dados e transformar empresas com inovação e inteligência.

Formação Multidisciplinar

O curso une programação, gestão, análise de dados, segurança e empreendedorismo. Você se torna capaz de atuar em equipes diversas, liderar projetos e resolver problemas reais do mercado.

Impacto na Sociedade

Profissionais de Sistemas de Informação criam aplicativos, sites, sistemas e plataformas que facilitam a vida das pessoas, melhoram processos e impulsionam negócios em todos os setores.

MULHERES NA TI

Por que é importante?

Promover a participação feminina em tecnologia fortalece diversidade de pensamento, inovação e resolução de problemas. Mulheres em TI trazem perspectivas variadas que aumentam a qualidade das soluções e melhoram a experiência dos usuários.

Iniciativas e Apoio

Programas de mentoria, grupos de estudo, bolsas e eventos específicos são formas eficientes de atrair e reter mulheres no curso e no mercado de trabalho. Empresas e universidades podem criar ambientes mais acolhedores e com políticas de equidade.

Oportunidades

As áreas de desenvolvimento, segurança, dados e gestão oferecem excelentes oportunidades. Incentivar estágios, projetos colaborativos e participação em hackathons ajuda na inserção profissional.

A IMPORTÂNCIA DA TECNOLOGIA

Inteligência Artificial

A IA está revolucionando setores inteiros, desde diagnósticos médicos até personalização de experiências digitais. Profissionais de BSI estão na vanguarda desta transformação.

Segurança Digital

Com a crescente digitalização, a proteção de dados e sistemas tornou-se crítica. Especialistas em segurança da informação são essenciais para empresas de todos os portes.

Computação em Nuvem

A migração para ambientes cloud permite escalabilidade, flexibilidade e redução de custos. Profissionais capacitados em arquiteturas cloud são altamente valorizados.

// Calculadora Interativa - JavaScript

let displayValue = '0'; // Valor exibido no visor

// Atualiza o visor da calculadora
function updateDisplay() {
    document.getElementById('display').innerText = displayValue;
}

// Adiciona número ou operador ao visor
function appendToDisplay(value) {
    const operators = '+-*/';
    const lastChar = displayValue.slice(-1);

    // Corrige botão de multiplicação visual
    if (value === '×') value = '*';

    // Evita operadores duplicados
    if (operators.includes(value) && operators.includes(lastChar)) {
        displayValue = displayValue.slice(0, -1) + value;
        updateDisplay();
        return;
    }
    // Evita múltiplos pontos decimais
    if (value === '.' && /\.\d*$/.test(displayValue)) return;

    // Substitui zero inicial (exceto ponto)
    if (displayValue === '0' && value !== '.') {
        displayValue = value;
    } else {
        displayValue += value;
    }
    updateDisplay();
}

// Limpa o visor
function clearDisplay() {
    displayValue = '0';
    updateDisplay();
}

// Realiza o cálculo da expressão
function calculate() {
    try {
        // Permite apenas números, operadores e parênteses
        if (/^[0-9+\-*/(). ]+$/.test(displayValue)) {
            // Avaliação segura
            const result = Function('"use strict";return (' + displayValue + ')')();
            if (Number.isFinite(result)) displayValue = String(result);
            else displayValue = 'Erro';
        } else {
            displayValue = 'Erro';
        }
    } catch (e) {
        displayValue = 'Erro';
    }
    updateDisplay();
}
